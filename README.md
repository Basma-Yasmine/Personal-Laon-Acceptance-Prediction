# Personal-Laon-Acceptance-Prediction

Project Overview:
This project predicts whether a bank customer is likely to accept a personal loan (deposit) offer using machine learning. The goal is to help banks identify high-potential customers and improve marketing efficiency.

Dataset
Source: Bank Marketing Dataset - Kaggle
Target Variable: deposit (yes/no)

Approach:
Performed exploratory data analysis on key features such as age, job, and marital status
Preprocessed categorical variables using label encoding

Trained and evaluated:
Logistic Regression
Decision Tree Classifier
Extracted business insights from model results
